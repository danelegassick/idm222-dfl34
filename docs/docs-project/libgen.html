Library Genesis

Library Genesis is a file sharing website for people to share educational materials such as textbooks and scientific articles. It is widely used amongst college students and those wishing to further their knowledge on a subject. It acts as a digital library in a sense.

Web Design

Responsive Design

Redesign

This project was for a user interface design class, the prompt being that I needed to redesign a website of choice. Simple enough. I had recently been to Library Genesis in order to get one of my textbooks and thought "Hey, this site kind of sucks!" So, I set out towards the redesign process.
